March 6, 2014

Neo4j LDAP plugin

A new project has been started on GitHub, called neo4j-ldap-plugin. This plugin is made for Neo4j (version 2.0.1) and enables user authentication using LDAP.

January 29, 2014

Data factory

A new project has been started on GitHub, called Data Factory. This Java library enables developers to create 'real data' to fill for instance a (demo) appication. It contains methods to create names, addresses and even texts in different languages.

The idea is to extend this library with a lot more data.

At the moment the data which can be generated are for Germany and The Netherlands. The generation of text is very simple and only available for Dutch.
